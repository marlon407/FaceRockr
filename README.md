# FaceRockr Coding Challenge

###Installation

Clone this project by terminal:
```bash
$ git@github.com:marlon407/FaceRockr.git
```

Go to project's folder:

```bash
$ cd FaceRockr
```

Installing dependencies
```bash
$ npm install
```

Run the server
```bash
$ npm start
```
